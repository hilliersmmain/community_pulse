# Screenshots\n\nThis directory contains screenshots of the Community Pulse dashboard.\n\n## Required Screenshots\n\n1. **dashboard_main.png** - Main dashboard view showing KPI metrics\n2. **cleaning_pipeline.png** - Data cleaning execution log\n3. **analytics_charts.png** - Interactive visualizations\n4. **before_after.png** - Data quality comparison\n\n## How to Capture\n\n1. Run the application locally: `streamlit run app.py`\n2. Generate sample data\n3. Run the cleaning pipeline\n4. Navigate through all tabs\n5. Take screenshots using your OS screenshot tool\n6. Save images with the filenames listed above\n\n## Image Requirements\n\n- Format: PNG\n- Resolution: At least 1920x1080\n- Clear, professional appearance\n- Include relevant UI elements
